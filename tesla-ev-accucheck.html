Tesla AccuCheck

Onafhankelijke accumeting voor Tesla Model 3, Model Y en Model S. AVILOO Flash Test op celniveau. Niet afhankelijk van Tesla-software of OTA-updates.

✓ Officieel AVILOO-partner ✓ TUV-gecertificeerd ✓ Op locatie bij de auto ✓ 16 jaar automotive ervaring

Tesla werkt met frequente OTA-updates die via internet direct worden doorgevoerd op de auto. Die updates kunnen de weergegeven accucapaciteit beïnvloeden zonder dat de accu zelf is veranderd. Een Tesla-getal dat vandaag 91% toont kan na een update morgen anders zijn. De AVILOO Flash Test staat volledig los van de Tesla-software en geeft altijd dezelfde onafhankelijke uitkomst, voor en na elke update.

Tesla OTA-updates en de SoH-waarde

AVILOO documenteerde in hun whitepaper dat een BMS-waarde na een reguliere software-update bij de dealer significant kan veranderen zonder dat de accu is aangeraakt. Bij Tesla is dit extra relevant omdat updates volledig automatisch worden doorgevoerd via internet, ook zonder bezoek aan een werkplaats. De AVILOO Flash Test analyseert elke cel individueel en is niet afhankelijk van welke software er op dat moment in de Tesla staat.

Typische degradatie

De Tesla Model 3 staat bekend om relatief lage degradatie. Na 5 jaar gemiddeld 85 tot 92% SoH bij normaal gebruik. Een van de meest gemeten modellen in de AVILOO database waardoor vergelijkingsdata zeer betrouwbaar is.

Bekende aandachtspunten

Tesla past via OTA-updates regelmatig de accubeheersoftware aan. Dit kan het weergegeven SoH-percentage beïnvloeden zonder dat de accu fysiek is veranderd. Intensief Superchargen, met name boven 80%, versnelt degradatie. Controleer laadgeschiedenis en gemiddeld laadpercentage.

Wat de AVILOO Flash Test controleert

Celanalyse van alle individuele cellen, onafhankelijk van Tesla-software en OTA-updates. Vergelijking met de grote Model 3 referentiedatabase in AVILOO. Geeft realistische context bij bouwjaar, kilometerstand en laadgedrag.

Typische degradatie

Vergelijkbaar met de Model 3. Na 3 jaar gemiddeld 90 tot 95% SoH bij normaal gebruik. Stabiele degradatiepatronen in de AVILOO database. De Model Y is inmiddels een van de meest verkochte EV's in Nederland.

Bekende aandachtspunten

Zelfde OTA-update risico als de Model 3. Tesla kan via een automatische update de weergegeven accucapaciteit wijzigen zonder bezoek aan een werkplaats. Vroege productiemodellen (2020-2022) hadden initieel hogere celspanningsafwijkingen die later via software zijn gecorrigeerd.

Typische degradatie

Oudere Model S modellen van voor 2017 kennen meer degradatie. Na 5 jaar gemiddeld 82 tot 90% SoH. Nieuwere modellen presteren stabiel. Groot accupakket degradeert trager in percentage maar kleine afwijkingen hebben grote impact op bereik.

Bekende aandachtspunten

Bij oudere Model S modellen kunnen individuele celgroepen significante afwijkingen vertonen die onzichtbaar zijn in het BMS-gemiddelde. Tesla heeft meerdere keren via OTA-updates de accubeheersoftware aangepast, wat de weergegeven capaciteit heeft beïnvloed. Bij een aankoopprijs van vaak meer dan €60.000 is een onafhankelijke meting essentieel.

Wat de AVILOO Flash Test controleert

Celanalyse van het volledige 100 kWh pakket, onafhankelijk van Tesla-software. Bijzonder waardevol bij oudere Model S modellen vanwege de leeftijd en de potentieel grote impact van celafwijkingen op bereik en restwaarde.
